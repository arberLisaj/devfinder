# Github user search 
Live Demo : https://arberlisaj.github.io/github-user-search/

##  Features
- 🔍 Search Github users and their details 
- ℹ   You can view the followers followings and repositories by clicking the number
- 🌐 Fetching data with axios
- 📱 Responsive Layout
